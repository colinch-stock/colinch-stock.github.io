# Colinch Stock 경량 하네스

- 공통 규칙은 상위 `AGENTS.md`를 따른다. 자동 포스팅 중 `CLAUDE.md` 전체를 읽지 않는다.
- 언어: 한국어 직접체 `~다`. 이론보다 실제 선택 기준과 계산 비교를 우선한다.
- 주제: ETF·주식, 연금저축·IRP·ISA, 청년 금융 상품, 절약·가계 재무, 투자 뉴스 해설.
- 반드시 날짜 기준, 수익률·세금·수수료·한도 같은 숫자 예시를 넣는다.
- 웹 검색: 금리, 제도, 상품 조건, 시장 뉴스가 핵심이면 1~2회 확인한다. 금융상품은 공식/금융사/공시 기준을 우선한다.
- SEO: 상품명, 계좌명, 세금·수익률·조건을 title·description·첫 문단에 넣는다.
- 태그 후보: `ETF`, `S&P500`, `KODEX`, `Tiger`, `배당ETF`, `해외ETF`, `주식투자`, `연금저축`, `IRP`, `ISA`, `퇴직연금`, `세액공제`, `청년도약계좌`, `사회초년생`, `파킹통장`, `예금`, `적금`, `금리`, `재테크`.
- 시각 자료: 수익률·세금 비교표, 월 납입 시뮬레이션 표, 상품 조건표, 간단한 누적 금액 그래프를 우선한다.
- 투자 권유처럼 쓰지 않는다. 원금 손실 가능성, 세금·수수료, 개인 상황 차이를 함께 적는다.
